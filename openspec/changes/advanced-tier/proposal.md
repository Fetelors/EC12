# Feature: Accommodation Tier with Pricing Display

## What are we building?
Add accommodation tier pricing display to the artist dashboard so artists can see what each tier costs before selecting.

## Why?
Artists need to see pricing for each tier to make informed choices. Currently they select blind — this causes support requests and last-minute changes that delay hotel bookings.

## Scope
- Display all 3 tiers with price and included features on the dashboard
- Show the artist's current selection highlighted
- Allow tier change (with a warning if hotel booking is already confirmed)
- Send an email notification to the advancing team when a tier changes

## Success Criteria
- All 3 tiers visible with prices before selection
- Current selection clearly highlighted
- Tier change blocked (or warned) if booking is locked
- Advancing team receives email on tier change
